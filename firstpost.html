Writing Your First Neural Network from Scratch

If you are interested in the field of artificial intelligence (AI), you have likely heard a lot about neural networks. These powerful algorithms are central to many AI systems and are able to learn and make predictions based on large amounts of data. However, if you are new to the world of AI, you may be wondering how to actually write a neural network. This task can seem intimidating, particularly if you do not have a background in computer science or machine learning. Don't be concerned â€“ writing your first neural network is easier than you may think! In this blog post, we will cover the basics of neural networks and provide a step-by-step guide for writing your first one.

First, let's define what a neural network is. A neural network is a type of machine learning algorithm that consists of interconnected "neurons." These neurons are organized into layers and are able to learn and make predictions based on the connections between them.

Imagine a neuron as a little computer inside your brain. It receives information from other neurons through tiny wires, processes that information, and sends it out to other neurons through more tiny wires. Together, all of these neurons work together to help your brain learn and make decisions. For example, when you see a beautiful flower, the neurons in your eyes send information about the flower to other neurons in your brain. These neurons then work together to help you understand what the flower is and how you should react to it. Neurons are a very important part of how your brain works!

In deep learning, a neuron is a unit that processes input data and generates an output. It is a fundamental building block of neural networks, which are algorithms that are designed to mimic the way the human brain processes information. Each neuron receives input data and combines it using a set of weights and biases. The resulting value is passed through an activation function, which determines the output of the neuron. The activation function is a mathematical function that takes in the input data and produces an output based on certain criteria. For example, the commonly used sigmoid activation function generates an output between 0 and 1, while the ReLU (Rectified Linear Unit) activation function generates an output that is either 0 or the input value, depending on whether the input is negative or positive. The output of the neuron is then passed on to other neurons in the network, where it can be further processed and used to make predictions or decisions. Neurons are organized into layers in a neural network, and the connections between them are used to transmit data and learn patterns in the input data. The structure and interconnections of the neurons in a neural network determine its behavior and capabilities.

Now that we have covered the basic theory, it's time to start coding!

To write a neural network, you'll need to have a dataset that you want your neural network to learn from. This dataset should be large and varied, and it should include examples of the type of data that you want your neural network to be able to handle.

Here, we are using the in-built MNIST dataset provided by the Keras library.

Import TensorFlow into your program to get started
                                    
import tensorflow as tf
Load the MNIST dataset
                                                    
mnist = tf.keras.datasets.mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

Once you have your dataset, you'll need to preprocess it to extract the relevant information and prepare it for training. This might involve cleaning and normalizing the data, splitting it into individual words or tokens, and performing other types of preprocessing to extract the relevant features. We are converting the sample data from integers to floating-point numbers

                                                    
x_train, x_test = x_train / 255.0, x_test / 255.0

Build a machine learning model

Next, you'll need to define the structure of your neural network. This will involve deciding how many layers it will have, how many neurons each layer will have, and how the neurons will be connected to each other.

"In this case, we are using the Sequential model provided by Keras in TensorFlow. You can read more about it at https://www.tensorflow.org/overview "
                                                    
    model = tf.keras.models.Sequential([
    tf.keras.layers.Flatten(input_shape=(28, 28)),
    tf.keras.layers.Dense(128, activation='relu'),
    tf.keras.layers.Dropout(0.2),
    tf.keras.layers.Dense(10)
    ])
                                                    
                                                
                                                
In order to write a neural network, you will need to consider the following factors:

    Input layer: The input shape, or the number of features, is required for the input layer.
    Hidden layers: You can add as many hidden layers as you like, but input variables are not necessary.
    Output layer: The number of nodes in the output layer should match the output. In classification, this means the number of classes, and in regression, it should be 1.
    Activation function: This function plays a crucial role in determining the output of the network.

Once you have your neural network structure defined, you can use a machine learning library, such as TensorFlow or PyTorch, to train your neural network on your dataset. This will involve iteratively adjusting the weights of the connections between the neurons in order to minimize the error between the predictions made by the neural network and the actual labels in your dataset.

Before you start training, configure and compile the model using Keras Model.compile. Set the optimizer class to adam, set the loss to the loss_fn function you defined earlier, and specify a metric to be evaluated for the model by setting the metrics parameter to accuracy.

Define a loss function for training using losses.SparseCategoricalCrossentropy, which takes a vector of logits and a True index and returns a scalar loss for each example.

                                                
loss_fn = tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True)
                                                
                                            
                                                
model.compile(optimizer='adam',

metrics=['accuracy'])

Once your neural network is trained, you can use it to make predictions on new data. This will involve providing your neural network with a set of input data and allowing it to generate output based on the patterns it has learned from your training dataset. Overall, writing your first neural network is a challenging but rewarding process that can give you a deep understanding of the capabilities and limitations of these powerful algorithms. With the right tools and resources, anyone can write their own neural network and start exploring the fascinating world of AI.
